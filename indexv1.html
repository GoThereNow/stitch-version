Finding the View Code Button

Follow these precise steps to access the developer output of any design component within the editor.

1

Select Canvas Item

Click any screen or component on the main canvas to focus the editor tools on that specific element.

2

Look Upward

Direct your gaze to the top toolbar of the Stitch interface where contextual actions appear.

3

Click View Code

Select the button labeled </> View Code to open the source panel for your selected design.
